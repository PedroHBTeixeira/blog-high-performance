# Introdução

Bem-vindo ao futuro do desenvolvimento web. Este artigo é escrito em **Markdown**, mas renderizado como HTML otimizado via Server Components.

## O que é MDX?

MDX permite usar componentes React dentro do Markdown. Veja este botão abaixo:

<Button>Clique Aqui (Botão React)</Button>

Isso não é uma imagem, é um componente interativo vivo dentro do texto!

## Dicas de Performance

1. Use **SSG** (Static Site Generation) para posts de blog.
2. Otimize imagens com `next/image`.
3. Use fontes locais.

> "A velocidade é a feature número um."
